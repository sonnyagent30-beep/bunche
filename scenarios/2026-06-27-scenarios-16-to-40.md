## SCENARIO 10: Error Alert (Workflow Failure)

**Customer:** Ada | **Status:** Payment confirmed → workflow fails → no IP delivered

---

### What happens

1. Ada pays ₦6,500 → Flutterwave webhook fires → Bunche receives payment
2. Workflow 2 (Payment Confirmation → IP Generation) starts
3. Workflow fails — Proxy-Seller API timeout, database error, or other technical issue
4. Customer is left hanging — paid but no IP
